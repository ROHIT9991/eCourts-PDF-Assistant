# eCourts-PDF-Assistant
A Python-based desktop application that streamlines PDF preparation for Indian eCourt e-Filing services. Built with Tesseract OCR, it enables advocates and attorneys to reorder pages, delete pages, rotate PDFs, perform OCR on scanned documents, and automate repetitive document preparation tasks before submission.
